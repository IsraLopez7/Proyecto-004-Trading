# Processed Data Directory

This directory will contain processed data files generated by the pipeline:

## Data Files

After running the pipeline, you will find:

### From `src/data_loader.py`:
- `data_clean.parquet` - Cleaned OHLCV data
- `data_train.parquet` - Training split
- `data_val.parquet` - Validation split
- `data_test.parquet` - Test split

### From `src/features.py`:
- `features.parquet` - All features combined
- `features_train.parquet` - Training features
- `features_val.parquet` - Validation features
- `features_test.parquet` - Test features
- `scaler.pkl` - StandardScaler fitted on train set

### From `src/labeling.py`:
- `labels.parquet` - All labels combined
- `labels_train.parquet` - Training labels
- `labels_val.parquet` - Validation labels
- `labels_test.parquet` - Test labels

### From `src/windows.py`:
- `windows_train.npz` - Training windows (X, y)
- `windows_val.npz` - Validation windows (X, y)
- `windows_test.npz` - Test windows (X, y)

### From `src/infer.py`:
- `signals_test.parquet` - Predicted signals for test set

### From `src/backtest.py`:
- `backtest_results.json` - Performance metrics
- `equity_curve.png` - Equity curve plot
- `returns_distribution.png` - Trade returns histogram

## Note

These files are automatically generated by running the pipeline scripts in order.
See the main README.md for the complete workflow.
